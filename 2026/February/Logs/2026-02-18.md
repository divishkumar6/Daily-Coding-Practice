### 1. Second Largest Element

- **Difficulty**: Easy
- **Topic**: Arrays
- **Solution**: `Week-1/Arrays/SecondLargestElement.java`
- **Approach**: Sort array and find first distinct element from end
- **Time Complexity**: O(n log n) - due to sorting
- **Space Complexity**: O(1) - in-place sorting

**Notes:**
- Sorted the array first for easier comparison
- Iterated from end to find first distinct element
- Handles duplicates by checking if adjacent elements are different
- Returns -1 if no second largest exists (all elements same)

**Possible Optimization:**
- Can be solved in O(n) time with single pass using two variables (max1, max2)
