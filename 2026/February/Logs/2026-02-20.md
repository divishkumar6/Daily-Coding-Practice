### 1. Remove Duplicates from Sorted Array

- **Difficulty**: Easy
- **Topic**: Arrays
- **Source**: LeetCode #26
- **Solution**: `Week-1/Arrays/RemoveDuplicatesFromSortedArray.java`
- **Approach**: Count unique elements, create temp array, copy back to original
- **Time Complexity**: O(n) - three passes through the array
- **Space Complexity**: O(k) - temporary array for k unique elements

**Notes:**
- First pass: Count unique elements to determine k
- Second pass: Store unique elements in temporary array
- Third pass: Copy unique elements back to original array
- Works correctly for in-place modification requirement
- Handles duplicates by comparing with previous element

**Current Solution Analysis:**
- ✅ Works correctly and passes all test cases
- ✅ Easy to understand and maintain
- ⚠️ Uses extra space O(k) for temporary array
- ⚠️ Makes three passes through the array

**Future Optimization Ideas:**
- Can be optimized to O(1) space using two-pointer technique
- Single pass solution possible by maintaining write pointer
- No need for temporary array - can modify in-place directly

**Optimization Approach (for future learning):**
```
Use two pointers:
- i: read pointer (iterate through array)
- j: write pointer (position for next unique element)
When nums[i] != nums[j], increment j and copy nums[i] to nums[j]
```
