### 2. Single Number (Easy) - LeetCode #136

**My Solutions:**

1. **Sorting** - O(n log n)
   - Sort and check pairs

2. **XOR (Optimal!)** - O(n), O(1) space ✨
   - XOR all numbers
   - Duplicates cancel out (a ^ a = 0)
   - Single number remains!

**Key Insight:** XOR properties make this elegant!
- a ^ a = 0
- a ^ 0 = a
- XOR is commutative
